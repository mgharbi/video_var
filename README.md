Motion Comparison - 2013
=================
Michael Gharbi <gharbi@mit.edu>

Prerequisites:
--------------
* CMake
* OpenCV
* Boost
* ffmpeg

Compilation:
------------
```
cd build  
cmake ..  
make  
```

Testing:
-------
```
cd build  
ctest
```

Running:
--------
The binaries are output in the `bin/` folder. 

Input data:
-----------
The files should go in the `data/` folder.

Output files:
-------------
The results are found in the `output/` folder.
